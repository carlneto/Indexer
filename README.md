# Indexer

Uma aplicação para macOS que indexa rapidamente ficheiros de uma pasta (ex.: TXT, PDF, Markdown, DOCX e imagens) e permite pesquisar por palavras, prefixos e padrões (regex), com pré‑visualização integrada via Quick Look.

## Descrição
O Indexer cria um índice local de palavras a partir de ficheiros no disco, suportando extração de texto (incluindo OCR para imagens comuns) e pesquisa rápida. A interface apresenta resultados e permite abrir/visualizar os documentos com um clique.

## Requisitos
- Sistema operativo: macOS 13.0 ou superior
- Xcode: 15.3 ou superior
- Swift: 5.10 ou superior

Nota: Este repositório inclui um pacote interno (UtilsPackage) com utilitários partilhados. Não é necessário instalar nada adicional — faz parte do projeto.
